## Attributes

| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| title	 | 聊天框标题	 | String	 | — | - |
| messages | 消息内容数组	 | Array |  - | - |
| footer | 发送框	 | slot |  - | - |


## Messages
| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| id	 | 消息唯一标识	 | String,Number	 | — | - |
| type | 消息内容类型	 | String | text,system,自定义标签（组件名）,renderEl| - |
| user | 用户信息	 | String |-| - |
| time | 发送时间	 | String  | -| - |
| content | 组件内容需要属性	 | Object  | -| - |
| position | left right	 | 发送者位置  | -| - |
| renderEl | 自定义渲染组件	 | Function  | -| - |




## User

| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| avatar	 | 用户头像链接	 | String	 | — | - |
