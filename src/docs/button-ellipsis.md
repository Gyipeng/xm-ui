## Attributes

| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| height	 | 	 | String	 | — | - |
| data | 按钮数据列表	 | Array |  - | 30 |

## Data
| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| name	 | 名字	 | String	 | — | - |
| color | 按钮类型	 | String | primary,success,warning,danger,info| - |
| 其余属性 | 请参考tag组件说明	 | String |-| - |
| slot | 插槽内容	 |  | -| - |
