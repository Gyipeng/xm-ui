## API

| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| title | 卡片主题名称| String | — | - |
| desc | 卡片描述 | String     | —  | - |
| actions | 卡片按钮名称 | Array   | — | - |
| src | 图片链接 音频链接 或者视频链接| String   | — | - |
| slot | 插槽 添加你的描述内容| String   | — | - |
| plain |朴素卡片| String  与size字段的mini一样  | — | 未来弃用 |
| other | 其他属性请移步媒体组件| String   | — | - |
| bodyStyle | 修改文字内容样式 | Object   | — | - |
| size | 卡片大小类型 | String   | 'medium', 'small', 'mini' | - |
