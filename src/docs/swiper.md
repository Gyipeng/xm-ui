## API

| 参数      | 说明          | 类型      | 可选值                           | 默认值  |
|---------- |-------------- |---------- |--------------------------------  |-------- |
| data | 数据源| Array | — | - |
| cardShowNumber | 每页显示的轮播数 | Numbers     | —  | 3 |
| centeredSlides | 轮播块是否居中 | String   | — | center |
| showPage | 是否一直显示箭头 | String   | — | - |


## Swiper事件

| 事件名称      | 说明          | 返回值  |
|---------- |-------------- |---------- |
| isShowArrow |  是否显示箭头 | -  |
| swiperClick | 关闭下拉 | - |
